# Трекер питания — установка как приложение

## Файлы
```
index.html      — основной файл приложения
manifest.json   — настройки PWA
sw.js           — service worker (офлайн-режим)
icons/          — иконки приложения (8 размеров)
```

## Шаг 1 — Создай репозиторий на GitHub

1. Зайди на https://github.com и войди в аккаунт (или зарегистрируйся)
2. Нажми **New** (зелёная кнопка)
3. Repository name: `nutrition-tracker`
4. Выбери **Public**
5. Нажми **Create repository**

## Шаг 2 — Загрузи файлы

1. В репозитории нажми **uploading an existing file**
2. Перетащи ВСЕ файлы и папку `icons` целиком
3. Нажми **Commit changes**

## Шаг 3 — Включи GitHub Pages

1. В репозитории зайди в **Settings**
2. В левом меню найди **Pages**
3. В разделе Source выбери **Deploy from a branch**
4. Branch: **main** → папка **/ (root)**
5. Нажми **Save**

Через 1–2 минуты появится ссылка:
`https://ТВОЙник.github.io/nutrition-tracker/`

## Шаг 4 — Установи на телефон

### iPhone (Safari):
1. Открой ссылку в Safari
2. Нажми кнопку «Поделиться» (квадрат со стрелкой вверх)
3. Выбери **На экран «Домой»**
4. Нажми **Добавить**

### Android (Chrome):
1. Открой ссылку в Chrome
2. Chrome автоматически предложит «Добавить на главный экран»
3. Или: меню (три точки) → **Добавить на главный экран**

После установки приложение открывается без браузерной строки,
работает офлайн и сохраняет данные между сессиями.

## Важно после загрузки на GitHub

В файле `sw.js` путь `/sw.js` должен соответствовать структуре.
Если репозиторий называется иначе чем `nutrition-tracker`,
замени в `index.html` строку:
```
navigator.serviceWorker.register('/sw.js')
```
на:
```
navigator.serviceWorker.register('./sw.js')
```
